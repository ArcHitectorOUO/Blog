    How to edit

    Edit everything in content.json
    1. Open portfolio/content.json. Replace placeholders with your details.
    2. To add a project, append an object to works with title, shortDesc, longDesc, tags, year, links, and highlight.
    3. Tags are freeform. Add any new tag inside a project's tags array; filter buttons update automatically.
    4. Set section visibility with the sections toggles (true/false).
